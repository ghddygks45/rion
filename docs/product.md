# RION 제품 개요

## 1. RION이란?

RION은 한국 주식시장에 특화된 주식 인텔리전스 서비스입니다.
오늘 어떤 섹터(테마)가 시장을 이끌고 있는지, 그 이유는 무엇인지 빠르게 파악할 수 있도록 돕습니다.

이 제품은 장 마감 무렵(종가 베팅)에 매매 결정을 내리는 직장인(30~50대)을 위해 설계되었습니다.

목표는 단순한 원시 데이터 제공이 아닌, 구조화된 의사결정 인사이트 전달입니다.

---

## 2. 타겟 사용자

- 한국 주식시장 참여자
- 시간이 부족한 직장인
- 종가 베팅 중심의 트레이더
- 주로 남성 (30~50대)
- 노이즈보다 명확성, 속도, 실행 가능한 인사이트를 선호하는 사용자

---

## 3. 핵심 가치 제안

- **오늘 돈이 몰리는 곳** 파악
- **특정 테마/섹터가 움직이는 이유** 설명
- **장 마감 전, 빠르고 구조적인 인사이트** 제공
- 의사결정 시간과 인지 부하 감소

---

## 4. 주요 기능

### 4.1 오늘의 테마 (목록)

- 아래 기준으로 종목을 섹터/테마별로 그룹화:
  - 거래대금
  - 상승률
- 각 테마에 포함되는 정보:
  - 섹터명
  - 뉴스 요약
  - 종목 목록 (종목명, 가격, 등락률, 거래대금)

---

### 4.2 테마 상세

- 테마 클릭 시 상세 화면
- 포함 정보:
  - 테마 개요 (LLM 생성 요약)
  - 최근 이슈/뉴스
  - 종목 테이블:
    - 종목명
    - 가격
    - 등락률
    - 거래대금
    - 주요 시그널 (예: 52주 신고가, 이상 거래량)

---

### 4.3 종목 상세

- 단일 종목 심층 분석

포함 정보:

- 뉴스 히스토리 (최근 관련 기사)
- 시장 개요:
  - 금, 유가(WTI), 나스닥 선물, S&P500 선물, KOSPI, KOSDAQ
- 오늘의 주요 이슈 (가장 많이 본 뉴스)
- 수급 현황 (기관/외국인)
- 차트 (임베드)
- 매매 전략 제안 (LLM 기반)
- 주요 시그널 / 이상 징후

---

### 4.4 알림

- 사용자가 등록 가능한 항목:
  - 종목
  - 시장 지표 (예: 유가)
- 알림 주기 설정 (5분, 10분 등)
- 가격 변동 또는 조건 기반 알림 수신
- 데스크톱 및 모바일 환경 모두 지원

---

## 5. 제품 철학

- 뉴스 앱이 아닙니다
- 소셜 플랫폼이 아닙니다
- **의사결정 지원 도구**입니다

핵심 원칙:

- 깊이보다 속도
- 양보다 명확성
- 노이즈보다 구조
- 원시 데이터보다 인사이트

---

## 6. UX 방향

- 사용자가 몇 초 안에 시장 흐름을 파악할 수 있어야 함
- 정보는 빠르게 스캔 가능해야 함
- 불필요한 인터랙션 없음
- "지금 중요한 것"을 우선 표시

---

## 7. 기술 방향 (개요)

- 데이터 수집:
  - 시장 데이터 (거래대금, 등락률)
  - 뉴스 크롤링 (네이버)
- 처리:
  - 테마 그룹핑
  - LLM 기반 요약
- 전달:
  - API 기반 백엔드
  - 실시간 알림 시스템

---

## 8. 성공 기준

- 사용자가 5~10초 안에 주도 테마를 파악할 수 있음
- 여러 기사를 읽지 않고도 "왜 시장이 움직이는지" 이해할 수 있음
- 사용자가 더 빠르고 자신 있게 매매 결정을 내릴 수 있음
