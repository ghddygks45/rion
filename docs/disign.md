# RION 디자인 가이드

## 1. 브랜드 개요

RION은 한국 주식시장 참여자를 위한 프리미엄 주식 인텔리전스 도구입니다.
디자인은 전문성, 신뢰감, 명확성을 반영해야 합니다.

이 제품은 가벼운 핀테크 앱이 아닙니다.
의사결정을 지원하는 진지한 도구입니다.

---

## 2. 디자인 톤

- 전문적
- 차분함
- 집중감
- 데이터 중심
- 프리미엄
- 다소 남성적이되, 공격적이지 않게

---

## 3. UI 철학

- 다크 모드 우선
- 밀도 높되 가독성 높게
- 구조적 레이아웃 (시각적 노이즈 없음)
- 빠른 스캐닝에 최적화
- 정보 계층 강조

---

## 4. 색상 시스템

### 기본 색상

- 배경: `#0A0A0A`
- 서피스: `#141414`
- 테두리: `#232323`

### 브랜드 색상

- 프라이머리: `#D4A017`
- 프라이머리 호버: `#E5B93C`

### 시장 색상 (한국 표준)

- 상승: `#E63B2E`
- 하락: `#2F6BFF`

### 강조색

- 액센트: `#FF6A00`

### 텍스트

- 기본 텍스트: `#E5E5E5`
- 보조 텍스트: `#A3A3A3`
- 비활성 텍스트: `#5A5A5A`

---

## 5. 색상 사용 규칙

- 추가 색상 도입 금지
- 파란색을 브랜드 색상으로 사용 금지
- 빨강은 반드시 상승을 의미
- 파랑은 반드시 하락을 의미
- 골드는 강조 용도로만 사용 (전체 UI에 남용 금지)
- 액센트 색상은 인터랙션용 (버튼, 활성 상태)

---

## 6. 레이아웃 원칙

- 카드 기반 구조 사용
- 일관된 간격 유지
- 복잡함 지양
- 요소를 엄격하게 정렬
- 장식보다 가독성 우선

---

## 7. 컴포넌트 가이드라인

### 카드

- 어두운 배경
- 은은한 테두리
- 최소한의 그림자
- 약간의 border-radius (과하게 둥글지 않게)

---

### 버튼

- 프라이머리: 액센트 또는 골드 (중요도에 따라)
- 세컨더리: 어두운 서피스 + 테두리
- 그라디언트나 화려한 효과 지양

---

### 테이블

- 고밀도, 고가독성
- 명확한 정렬
- 색상은 중요 시그널(상승/하락)에만 사용
- 과도한 스타일링 지양

---

### 차트

- 어두운 배경
- 빨강 = 상승
- 파랑 = 하락
- 최소한으로, 기능 중심으로

---

## 8. 타이포그래피

- 폰트: Pretendard
- 깔끔하고 모던하게
- 강한 계층 구조
- 높은 대비
- 빠른 스캐닝에 최적화
- 장식적 폰트 지양

---

## 9. 인터랙션 원칙

- 빠른 반응
- 애니메이션 최소화
- 불필요한 전환 효과 없음
- 미관보다 사용성 집중

---

## 10. 금지 사항

- 가볍고 놀이 같은 핀테크 스타일 금지
- 파스텔 색상 금지
- 과도하게 둥근 UI 금지
- 반짝이거나 화려한 비주얼 금지
- 카지노 같은 골드 남용 금지
- SNS 스타일 UI 금지

---

## 11. 구현 규칙

생성되는 모든 UI는 이 디자인 가이드를 엄격히 따라야 합니다.

창의성과 일관성이 충돌하는 경우:
→ 항상 일관성을 우선합니다
